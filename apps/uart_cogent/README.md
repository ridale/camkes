# UART driver in Cogent

Prototype cogent UART driver for sabre.  Need to have cogent installed and repository
checked out in `./tools/cogent`.

## To build

```sh
mkdir build
cd build
../init-build.sh -DCAMKES_APP=uart_cogent -DPLATFORM=sabre
ninja

```
